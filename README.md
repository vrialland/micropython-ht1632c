# MicroPython lib for HT1632C led matrix from Sure Electronics

## Wiring

| ESP8266 | HT1632C | Color  | GPIO |
|---------|---------|--------|------|
| D5      | DATA    | Blue   | 14   |
| D6      | CS      | Green  | 12   |
| D7      | WR      | Yellow | 13   |
| D8      | CLK     | Orange | 15   |
